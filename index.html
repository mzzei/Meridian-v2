Bem-vindo ao Meridian

A chave API da Anthropic é um código pessoal que libera o agente para usar inteligência artificial — sem ela, o agente não consegue analisar partidas nem responder perguntas.

Criar uma é gratuito e leva menos de 2 minutos: clique em "Configurar agora", cole o código e pronto.

Como analisar

Dois modos de entrega

O Meridian escolhe o formato pelo que você digita. Entenda a diferença para evitar card “errado”.

Análise padrão

Relatório completo no card com abas fixas — o mesmo modelo do export PDF:

  • Resumo · fatores, tendências, tickets
  • Tática · probabilidades, gols esperados, técnicos
  • Desempenho · jogadores-chave e stats
  • Cartões & Faltas · disciplina
  • Escanteios · leitura e mercados (PDF de referência)
  • Escalação · mapa de campo / prováveis
  • Dados Avançados · placares, incerteza, lacunas

Como pedir: Flamengo x Palmeiras ou analise Brasil x Argentina (confronto claro Time A × Time B).

Chat livre

Card flexível (abas livres) ou texto — para opinião, recortes e pós-jogo sem relatório completo.

Exemplos: “o que achou do 2º tempo?”, “só os gols do jogo”, “opinião sobre o jogo de hoje”.

Contexto

Se a pergunta for ambígua (ex.: “jogo de hoje” sem times), o app abre um popup de opções — não inventa o jogo.

Você também pode fixar pano de fundo com o ícone de lápis/contexto ao lado da caixa de mensagem (ao lado do clipe de anexo).

Configurações avançadas
Desliga o desfoque de fundo (blur/vidro fosco) e as animações decorativas — deixa a interface mais leve para computadores que engasgam. O app já detecta automaticamente quando o dispositivo está renderizando abaixo de 45fps e liga isso sozinho; use este botão para forçar manualmente.
Informações avançadas (protegidas por senha)
Proxy grátis (Cloudflare) para Anthropic, API-Football, football-data.org e FPL: a chave não precisa ficar no browser e resolve o bloqueio (CORS) que impede essas APIs de funcionar direto no navegador. Código e passo a passo em worker/ no repositório.
Cobertura da coleta (A/B/C): A campeonato = ESPN grátis (+ football-data.org via Worker se secret/chave); B time = técnico via API-Football (Free não cobre temporada 2026 em fixtures — escalação confirmada só por busca); C analítico = web_search e, na EPL, FPL via Worker. AF e FD não funcionam no browser sem Worker (CORS).
