# Assets

## wc-trophy.png

Imagem da Taça da Copa do Mundo usada no emblema (logo) da sidebar.

- Salve o arquivo exatamente como **`wc-trophy.png`** nesta pasta.
- Pode ter fundo preto/escuro — o CSS usa `mix-blend-mode: screen`, que faz
  o fundo escuro desaparecer sobre o verde do emblema. Fundo transparente
  também funciona.
- Recomendado: imagem quadrada ou vertical, taça centralizada, lado ≥ 256px.

Enquanto este arquivo não existir, o emblema mostra automaticamente uma taça
em SVG vetorial como fallback (o `<img>` se auto-remove via `onerror`).
